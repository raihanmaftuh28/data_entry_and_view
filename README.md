# Data entry and view

Front-end and backend suite to demonstrate how to do data entry both front-end and back-end wise


## Getting Started

### Prerequisites
- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/raihanmaftuh28/data_entry_and_view.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
